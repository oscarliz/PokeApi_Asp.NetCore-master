¡Hola, Solvex Dominicana!

Bienvenido a su nueva aplicación de una sola página, construida con:

  • ASP.NET Core y C# para código multiplataforma del lado del servidor
  • Angular y TypeScript para el lado del cliente código
  • Bootstrap para diseño y estilo

Para ayudarlo a comenzar, también configuramos:

  • Angular CLI integration. Se ejecuta en segundo plano de forma automática, por lo que los recursos del lado del cliente se crean dinámicamente a pedido y la página se actualiza cuando modifica cualquier archivo.
  • Efficient production builds. En el modo de producción, las funciones de tiempo de desarrollo están deshabilitadas y su configuración de publicación de dotnet invoca automáticamente ng build para producir archivos JavaScript compilados con anticipación y minimizados.

El subdirectorio ClientApp es una aplicación CLI estándar de Angular. Si abre un símbolo del sistema en ese directorio, puede ejecutar cualquier comando ng (por ejemplo, ng test ), o usar npm para instalar más paquetes en él.
